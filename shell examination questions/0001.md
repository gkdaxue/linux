## 题目要求

        每天生成一个磁盘使用情况的日志文件(.log结尾) 并放到 /data/disklog 下, 如 2008-08-08.1og 文件, 写出 shell 脚本即可.

## 参考答案

```bash
#!/bin/bash
d=$(date '+%F')
logfile=${d}.log
dir=/data/disklog
if [ ! -e ${dir} ]; then
  mkdir -p ${dir}
fi
df -h >> ${dir}/${logfile}
```

## 答案解析

```bash
#!/bin/bash
## 获取当前日期
d=$(date '+%F')

## 获取当前日志文件名称, 格式为 2008-08-08.log
logfile=${d}.log

## 检查日志目录是否存在
dir=/data/disklog
if [ ! -e ${dir} ]; then
 mkdir -p ${dir}
fi


df -h >> ${dir}/${logfile}
```
